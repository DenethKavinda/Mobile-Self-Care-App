# Daily Selfcare

Daily Selfcare is a Kotlin-based Android app designed to help users build healthier daily routines and stay consistent with personal wellness goals.

## Features

- User login and sign-up flow
- Water intake tracking with progress display
- Daily habit tracking
- Mood logging
- Journaling for quick personal notes
- Breathing exercise screen
- Wellness tips section
- Profile and settings screens
- Local data storage using SQLite and shared preferences

## Tech Stack

- Kotlin
- Android SDK
- AndroidX
- Material Design
- SQLite
- SharedPreferences

## Project Structure

- `MainActivity` - app entry screen
- `LoginActivity` and `SignUpActivity` - authentication flow
- `HomeActivity` - dashboard and navigation hub
- `WaterTrackerActivity` - hydration tracking
- `HabitsActivity` - habit management
- `MoodActivity` - mood tracking
- `JournalActivity` - journal entries
- `BreathingActivity` - breathing exercises
- `TipsActivity` - self-care tips
- `SettingsActivity` and `ProfileActivity` - user settings and profile management

## Getting Started

1. Open the project in Android Studio.
2. Let Gradle sync and download dependencies.
3. Run the app on an emulator or connected Android device.

## Requirements

- Android Studio
- JDK 11 or later
- Android device or emulator

## Notes

- The app currently stores data locally on the device.
- The minimum SDK is 24 and the target SDK is 36.
